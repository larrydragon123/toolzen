首页布局方案

首页是流量分发枢纽，决定了用户能否快速找到所需工具，也直接影响SEO

A

分类网格 + 顶部搜索 (推荐)

搜索栏 + 分类卡片 + 热门工具单独入口。每个工具独立页面，URL结构 /tools/json-formatter

B

单页All-in-One (SPA工具集)

左侧工具列表，右侧为当前工具面板。所有工具在一个页面内切换。对SEO不友好(只有一个URL)

C

工具+BLOG混合站

工具页 + 博客文章(SEO教程、工具使用指南)，用内容带动工具流量

URL 结构设计 (SEO关键)

/ ← 首页 (分类网格)
/json-formatter/ ← JSON格式化工具
/base64-encode-decode/ ← Base64编解码
/mortgage-calculator/ ← 房贷计算器
/blog/seo-basics-2026/ ← 博客文章 (模块3时添加)
/compare/vscode-vs-cursor/ ← 工具对比页 (模块3)
